## 🇬🇧 English

This project is a **simple console-based Tic Tac Toe (XOX) game** developed in Java.  
The game is designed for two players who take turns placing their symbols (X and O) on a 3x3 board.

The board is implemented using a standard Java array, and the game logic checks for winning and draw conditions after each move.

The purpose of this project is to practice:

- Basic Java syntax  
- Arrays  
- Loops and conditional statements  
- Game flow control in console applications  

The game ends when:

- A player completes a row, column, or diagonal  
- The board is full and results in a draw

## 🇹🇷 Türkçe

Bu proje, Java ile geliştirilmiş **basit bir konsol tabanlı XOX (Tic Tac Toe) oyunudur**.  
İki oyuncu sırayla 3x3 tahtaya X ve O sembollerini yerleştirir.

Oyun tahtası Java dizisi (array) kullanılarak oluşturulmuştur ve her hamleden sonra kazanma veya beraberlik durumu kontrol edilir.

Projenin amacı:

- Temel Java sözdizimini pekiştirmek  
- Array (dizi) kullanımını öğrenmek  
- Döngü ve koşul yapılarıyla oyun mantığı geliştirmek  
- Konsol uygulama akışını yönetmek  

Oyun şu durumlarda sona erer:

- Bir oyuncu satır, sütun veya çapraz tamamladığında  
- Tahta dolduğunda ve beraberlik oluştuğunda
